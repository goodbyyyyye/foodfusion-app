FoodFusion app
   FEATURES
User Side:
Browse food items
Search food
Filter by categories
Add to cart with quantity
View cart total
Place orders (COD / PAYPAL)
View order status (Pending → Preparing → On the Way → Delivered)
Order history

Admin Side:
Add / edit / delete food items
Create and manage categories
Assign items to categories
View all user orders
Update order status

Frontend:
